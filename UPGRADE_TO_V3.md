# Upgrade to v3 — Инструкция по обновлению

## Что нового в v3

- **Новая категория техники** — «Юк транспорт» (грузовики, самосвалы, бензовозы)
- **Раздел «Камчиликлар»** — ввод недостатков и комментариев операторов
- **Кнопка ✕ удаления строки** — если оператор случайно добавил лишнюю строку работ
- **Исправлен баг расчёта суммы** — колесо мыши больше не изменяет значения в полях
- **Excel-отчёт полностью переделан** — 8 листов, точно как у операторов
- **Кнопки редактирования** во всех справочниках
- **13 организаций по умолчанию** (добавлены Чорва, ДЗЗ Латтико, Азтекс)

## Обновление на сервере — пошагово

### ⚠️ ВАЖНО: Сначала сделайте резервную копию базы!

Откройте CMD от администратора:

```
cd C:\transport-report
```

```
copy instance\transport.db instance\transport.db.backup
```

### Шаг 1 — Остановите сервис

```
nssm.exe stop TransportReport
```

### Шаг 2 — Замените файлы программы

Распакуйте содержимое `transport-report-v3.zip` поверх папки `C:\transport-report\`,
соглашаясь на замену всех файлов.

**НЕ удаляйте:**
- `instance\transport.db` (ваша база данных)
- `nssm.exe`
- `logs\` (логи)

### Шаг 3 — Запустите миграцию базы

```
"C:\Program Files\Python314\python.exe" migrate_to_v3.py
```

Скрипт:
- Создаст новую таблицу для недостатков
- Автоматически переклассифицирует грузовики (HOWO, MAN, Камаз, Isuzu, Бензовоз) из категории «Юкори» в новую категорию «Юк транспорт»
- Покажет итоговую статистику

**Пример вывода:**
```
Step 1: Creating deficiencies table...
  OK: deficiencies table ready

Step 2: Reclassifying trucks as Yuk transport...
  Reclassified: 23 equipment
  Examples:
    - HOWO 80 175 ОАА
    - MAN 80 254 ОАА
    ...

Step 3: Final statistics:
  MTZ Tractors             : 22
  High-perf & special      : 55
  Yuk transport (trucks)   : 28

OK: Migration completed successfully!
```

### Шаг 4 — Запустите сервис

```
nssm.exe start TransportReport
```

```
nssm.exe status TransportReport
```

Должно быть `SERVICE_RUNNING`.

### Шаг 5 — Проверьте работу

Откройте `http://localhost:5050` в браузере.

Проверьте что:
- Старые данные на месте (зайдите на любую дату и посмотрите)
- В меню появился раздел **«Камчиликлар»**
- В справочнике техники появилась кнопка ✏ редактирования
- При добавлении техники в категориях есть «Юк транспорт»
- На странице ввода данных (Маълумот киритиш) появилась секция **«Юк транспорт»** (если в организации есть такая техника)
- Колесо мыши в полях миқдор/нарх больше не меняет значения

## Если что-то пошло не так

### Откат к v2

```
nssm.exe stop TransportReport
```

Восстановите базу:
```
copy instance\transport.db.backup instance\transport.db
```

Распакуйте старый ZIP v2 поверх файлов и запустите:
```
nssm.exe start TransportReport
```

### Проверка логов

```
type C:\transport-report\logs\error.log
```

Если там будут Python-ошибки — пришлите их, разберём.

## Новые возможности в работе

### Как работать с «Камчиликлар» (Недостатки)

1. В меню выберите **«Камчиликлар»**
2. Выберите дату
3. Введите текст недостатка в форму «Янги камчилик қўшиш»
4. (Необязательно) выберите организацию
5. Нажмите «Қўшиш»
6. Все добавленные недостатки появятся в Excel-отчёте на листе **«ОБШИЙ КАМЧИЛИК»**

### Как удалить лишнюю строку работ

Если в карточке техники вы случайно нажали «+ Қўшимча иш қўшиш» или добавили неправильную строку:
- Нажмите красную кнопку **✕** справа от строки
- Программа спросит подтверждение
- Строка удалится (если это не единственная строка — иначе поля очистятся)

### Как изменить запись в справочнике

Во всех справочниках (Ташкилотлар, Техникалар, Иш турлари, Буюртмачилар):
- Нажмите кнопку **✏** напротив записи
- Строка превратится в форму редактирования
- Внесите изменения и нажмите «Сақлаш»
- Для отмены — кнопка «Бекор»

### Как добавить технику «Юк транспорт»

1. Ташкилотлар → Техникалар
2. В форме «Янги техника қўшиш» выберите категорию **«Юк транспорт»**
3. Укажите тип (Самосвал / Грузовой / Бензовоз / Спецтехника)
4. Заполните остальные поля и нажмите «Қўшиш»

## Структура нового Excel-отчёта (8 листов)

1. **ОБШИЙ** — общая сводка по всем организациям: 3 типа оплаты × 3 категории техники
2. **ОБШИЙ КАМЧИЛИК** — все недостатки за день
3. **СВОД МТЗ** — сводка по МТЗ тракторам в разрезе организаций
4. **Свод ичи МТЗ** — детализация: каждый трактор, вид работы, сумма
5. **СВОД Юкори** — сводка по высокопроизводительной технике
6. **Свод ичи Юкори** — детализация
7. **СВОД Юк транспорт** — сводка по грузовикам (НОВЫЙ!)
8. **Свод ичи Юк транспорт** — детализация (НОВЫЙ!)

Все листы готовы к печати — альбомная ориентация A4, автоподгонка ширины, повторяющиеся заголовки на каждой странице.
